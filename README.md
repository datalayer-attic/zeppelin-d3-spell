# D3.js for Apache Zeppelin 

D3.js Spell Visualization for Apache Zeppelin.

## Example

Type some SVG in a paragraph and click on `run`.

```
%d3

<svg width"="50" "height"="50">
  <circle cx="50" cy="50" r="50" style="fill: yellow" />
<svg>
```
